# Transformation of sequences
우리가 소비하는 순서의 값이 항상 필요한 형식이 아닙니다. 때로는 데이터에 너무 많은 노이즈가있어 값을 제거합니다. 때로는 각 값을 더 풍부한 객체 또는 더 많은 값으로 확장해야 할 수도 있습니다. 연산자를 작성함으로써, Rx는 사용자가 관찰 할 수있는 시퀀스의 값의 양과 품질을 제어 할 수 있습니다.
